Bắt đầu với Blink

Chương trình đầu tiên của mọi lập trình viên Arduino. Chỉ với vài dòng code, bạn đã có thể điều khiển một đèn LED nhấp nháy liên tục.

  • ✔️ Cấu trúc đơn giản (setup và loop)
  • ✔️ Dễ dàng hiểu được chỉ sau 5 phút
  • ✔️ Cầu nối chuyển tiếp sang các thiết bị phức tạp
Blink.ino 

// Hàm setup chạy 1 lần khi khởi động
void setup() {
  // Cài đặt chân 13 làm đầu ra
  pinMode(13, OUTPUT);
}

// Hàm loop chạy lặp lại liên tục
void loop() {
  digitalWrite(13, HIGH);  // Bật sáng LED
  delay(1000);             // Đợi 1 giây
  digitalWrite(13, LOW);   // Tắt LED
  delay(1000);             // Đợi 1 giây
}
